# PopX UI

A pixel-perfect mobile UI implementation of the PopX app built using **React**, **Tailwind CSS**, and **React Router**.  
The mobile interface is centered on the webpage and closely follows the provided design.

---

##Live Features

-  Mobile-first UI centered on desktop
-  Pixel-perfect design using Tailwind CSS
-  Smooth navigation with React Router
-  Interactive UI (hover, cursor states)
-  Accessible form inputs and buttons

---

## Tech Stack

- **React JS**
- **Tailwind CSS**
- **React Router DOM**
- **Vite**
